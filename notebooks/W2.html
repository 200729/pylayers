Interactive session illustrating UWB Ray Tracing Simulation¶

A set of UWB simulations has been performed in the environement of SIRADEL. This section describes some available features of the developped tool.

In[*]:

1 Starting the simulation¶

The simulation is loaded by reading a configuration file which contains all the required informations about the simulation parameters as well as the name of the data file which has already been simulated.

The simulation is as follow :

[files]
at = simul9.mat
tx = w2tx.ini
slab = simul9.slab
txant = UWB_CEA.vsh3
rx = w2rx.ini
patra = simul9.patra
conf = $HOME/Pyproject/project.conf
palch = simul9.palch
struc = Lstruc.str
rxant = UWB_CEA.vsh3

[tracing]
maxdeep = 10
distdiff = 5.0
var2d3d = 1

[waveform]
tw = 30
band = 0.499
fc = 4.493
thresh = 3
fe = 50
type = generic

[frequency]
fmax = 11
nf = 181
fmin = 2

Initialization¶

In[2]:

from pylayers.antprop.coverage import *
import time

Instantiate a coverage object. By defaut, the TA-Office.str layout strucure is loaded.

This information can be modified in the coverage.ini file in the project directory.

In[3]:

C=Coverage()
C.L.filename

Out[3]:

'TA-Office.str'

Coverage¶

Coverage required Layout graphs.

  • If the graphs for the given layout have already been built, they are loaded from the project directory.
  • Otherwise, they are built (it may take few minutes ) and stored into the project directory for the next execution.

In[4]:

try :
    C.L.dumpr()
except:
    C.L.build()

Then, the coverage calculation can be launched

In[5]:

C.cover()

You can display the coverage result with

In[6]:

C.showPower()

The transmitter coordinates are :

In[7]:

C.tx

Out[7]:

array([-10,  10])

This can be modified, and the coverage can be updated

In[8]:

C.tx = np.array((30,12))
C.cover()
C.showPower()

Speed is for all Tx positions on the grid

In[9]:

C.tx = np.array((39,1))
start = time.time()
C.cover()
finish = time.time()
C.showPower()
print 'in %1.2f seconds' % (finish-start)

in 8.18 seconds

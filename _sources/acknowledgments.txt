
Third party packages 
====================

Pylayers is written in Python and is standing on the shoulders of the
following giant Python packages:

* `numpy <http://www.numpy.org>`_ ( for algebra in multi dimensions )  
* `scipy <http://www.scipy.org>`_ ( for scientific libraries )
* `matplotlib <http://matplotlib.org/gallery.html>`_ ( for the graphical outputs ) 
* `matplotlib basemap <http://matplotlib.org/basemap>`_ ( for GIS ) 
* `pandas <http://pandas.pydata.org>`_ ( for time series and data manipulation )
* `SimPy <http://simpy.sourceforge.net>`_ ( for discrete events simulation ) 
* `networkx <http://networkx.github.io>`_ ( for graph description and algorithms ) 
* `shapely <http://toblerity.github.io/shapely/project.html>`_ ( for GIS and planar geometry )   
* `ipython <http://ipython.org>`_ ( to rule them all ... ) 


Pylayers relies also on the interactive 3D viewing program `geomview <http://www.geomview.org>`_ tool for fast and simple interaction with 3D entities.

Download and Installation
=========================

The current version is tagged 0.1. 

Download the last release in your preferred format on github.

.. code-block:: bash
    
    $ git clone https://github.com/pylayers/pylayers.git


We warmly encourage all new user to contribute new suggestions, algorithms, models and other improvements back to the project

`**Just fork it !** <http://github.com/pylayers/pylayers/fork>`_ on your
github account and get in touch on the `mailing list <https://groups.google.com/forum/#!forum/pylayers-users>`_
if you have some interest or see practical applications.

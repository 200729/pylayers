Citing Pylayers¶

If you use pylayers in scientific publication, we would appreciate citations to the following paper:

PyLayers : An Open Source Dynamic Simulator for Indoor Propagation and Localization, N. Amiot, M.Laaraiedh,B.Uguen,Communications Workshops ICC 2013

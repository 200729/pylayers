Parameters :

filefield : string

transpose : boolean

default False

Examples

>>> from pylayers.simul.simulem import *
>>> S = Simul()
>>> S.load('default.ini')
>>> C = Ctilde()
>>> C.load(pyu.getlong(S.dtud[1][1],'output'))

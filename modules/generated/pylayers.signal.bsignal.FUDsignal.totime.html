pylayers.signal.bsignal.FUDsignal.totime¶

FUDsignal.totime(Nz=1, ffts=0)[source]¶

transform to TUDsignal

Parameters:

Nz : int

Number of zeros for zero padding

ffts : nt

fftshift indicator (default 0 )

See also

FUsignal.ift

Examples

>>> import pylayers.simul.link as lk

>>> L = lk.DLink(verbose=False)
>>> aktk = L.eval()
>>> H = L.H

>>> f,a = H.show(cmap='jet')

We keep only the significant rays

>>> H.cut()
>>> f,a = H.show(cmap='jet')

We then transform into the time domain

>>> T1 = H.totime()
>>> T1.plot()

>>> H.minphas()
>>> T2 = H.totime()
>>> T2.plot()

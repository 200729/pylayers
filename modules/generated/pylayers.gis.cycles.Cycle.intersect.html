pylayers.gis.cycles.Cycle.intersect¶

Cycle.intersect(Cy)[source]¶

Find intersection with an other cycle

Parameters:

Cy : Cycle
Returns:

flip : boolean

indicates the orientation of the common brin True : flipped direction False : same direction

brin : common portion of th

Notes

flip,path = C1.intersect(C2)

if flip == True –> The 2 cycles have a reverse travel direction for the common path

path is the common path along the cycle

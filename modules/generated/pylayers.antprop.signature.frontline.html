pylayers.antprop.signature.frontline¶

pylayers.antprop.signature.frontline(L, nc, v)[source]¶

determine cycle frontline

This function calculates the scalar product of the normals of a cycle and returns the indev of segments whith are facing the given direction v. scalar product < 0.

Parameters:

L : Layout

nc : cycle number

v : direction vector
Returns:

nsegf : list

pylayers.antprop.rays.Rays.mirror¶

Rays.mirror(H=3, N=1)[source]¶

mirror a ray termination

Parameters:

H : float

ceil height (default 3m)

N : int

handle the number of mirror reflexions
Returns:

d : dict

k : zm v: alpham k : zp v: alphap

Examples

>>> ptx = np.array([1,1,1.5])
>>> prx = np.array([2,2,1.2])
>>> r = Rays(ptx,prx)
>>> d = r.mirror()
>>> d[-1.5]
array([ 0.55555556])

pylayers.network.network.Network.Gen_tuple¶

Network.Gen_tuple(gene, wstd, var)[source]¶

generate a specific tuple

Parameters:

gene : tuple(x,y) iterator

wstd : str

var : list

len(var) = len(gene)

Examples

>>> from pylayers.network.network import *
>>> N=Network()
>>> tup = zip(range(5),range(5))
[(0, 0), (1, 1), (2, 2), (3, 3), (4, 4)]
>>> g = iter(tup)
>>> wstd='string wstd'
>>> var=[10,11,12,13,14]

>>> T=N.Gen_tuple(g,wstd,var)
>>> T.next()
(0, 0, 'string wstd', 10)
>>> T.next()
(1, 1, 'string wstd', 11)

pylayers.signal.bsignal.FUsignal.energy¶

FUsignal.energy(axis=-1, Friis=False, mode='mean')[source]¶

calculate energy along a given axis of the FUsignal

Parameters:

axis : (default 0)

Friis : boolean

mode : string

mean (default) | center | integ | first | last

Notes

If Friis is true energy is multiplied by

\(\frac{-j*c}{4 pi fGHz}\)

axis = 0 is ray axis

if mode == ‘mean’

\(E=\frac{1}{K} \sum_k |y_k|^2\)

if mode == ‘integ’

\(E=\delta_x \sum_k |y_k|^2\)

if mode == ‘center’

\(E= |y_{K/2}|^2\)

Examples

>>> ip = TUsignal()
>>> ip.EnImpulse()
>>> En1 = ip.energy()
>>> assert((En1>0.99) & (En1<1.01))

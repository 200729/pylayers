pylayers.network.network.Network._get_SubNet¶

Network._get_SubNet(wstd=[])[source]¶

get SubNetworks of a network

Parameters:wstd : specify which SubNet to create

Warning

ALWAYS use self._get_wstd() BEFORE !

Examples

>>> from pylayers.network.network import *
>>> N=Network()
>>> for i in range(2):
        no = Node.Node(ID=i,wstd=['wifi','bt'])
        N.add_nodes_from(no.nodes(data=True))

>>> no = Node.Node(ID=2,wstd=['wifi'])
>>>    N.add_nodes_from(no.nodes(data=True))
>>> N._get_wstd() # VERY IMPORTANT 

>>> N._get_SubNet()
>>> N.SubNet['bt'].nodes()
[0, 1]
>>> N.SubNet['wifi'].nodes()
[0, 1, 2]

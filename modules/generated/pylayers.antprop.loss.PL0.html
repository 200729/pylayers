pylayers.antprop.loss.PL0¶

pylayers.antprop.loss.PL0(fGHz, GtdB=0, GrdB=0, R=1)[source]¶

Path Loss at frequency fGHZ @ R

Parameters:

fGHz: float

frequency GHz

GtdB: float

transmitting antenna gain dB (default 0 dB)

GrdB: float

receiving antenna gain dB (default 0 dB)

R : float

distance in m
Returns:

PL0 : float

path @ R

Notes

\[PL_0 = -20 log_{10}(\frac{\lambda}{4\pi}) - GtdB -GrdB\]

Examples

>>> fGHz  = 2.4
>>> PL = PL0(fGHz)
>>> assert (PL<41)&(PL>40),"something wrong"

pylayers.util.pyutil.encodmtlb¶

pylayers.util.pyutil.encodmtlb(lin)¶

encode python list of string in Matlab format

Parameters:

lin : input list

encodmtlbi(lin) : :
Returns:

lout : output list

Notes

The List is read column by column and written line by line in a same NxM matrix. If char does not exist it is replaced by space.

Examples

>>> lin = ['aaa','bbbbbbb','ccc','dd']
>>> F   = {}
>>> F['lin']=encodmtl(lin)
>>> print F['lin']
>>> io.savemat('encodmtlb_ex.mat',F)

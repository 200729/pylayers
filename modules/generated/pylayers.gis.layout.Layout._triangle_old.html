pylayers.gis.layout.Layout._triangle_old¶

Layout._triangle_old(poly_surround, poly_holes=[], mesh_holes=False)[source]¶

perfome a delaunay partitioning on shapely polygons

Parameters:

poly_surround : sh.Polygon

A single polygon to be partitionned

poly_holes
: list of sh.Polygon

A list of polygon contained inside poly_surround. they are considered as holes

mesh_holes
: bool

If True make the delaunay partition of poly_holes else : only partitioning poly_surround and traits poly_holes as holes

Notes

uses triangle library

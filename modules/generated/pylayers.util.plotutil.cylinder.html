pylayers.util.plotutil.cylinder¶

pylayers.util.plotutil.cylinder(fig, pa, pb, R)[source]¶

plot a cylinder

Parameters:

pa : np.array 3 x nc

pb : np.array 3 x nc

R : np.array 1 x Nc

Examples

>>> from pylayers.util.plotutil import *
>>> import numpy as np
>>> pa = np.array([0,0,0])
>>> pb = np.array([0,0,10])
>>> fig = plt.figure()
>>> cylinder(fig,pa,pb,3)
>>> plt.show()

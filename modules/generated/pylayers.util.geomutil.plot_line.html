pylayers.util.geomutil.plot_line¶

pylayers.util.geomutil.plot_line(ax, ob, color='#999999')[source]¶

plot line

Parameters:

ax : matplotlib axes

ob : shapely object

color : string

default ‘#999999’

Notes

color = v_color(ob)

References

Shapely

Examples

>>> from pylayers.util.geomutil import *
>>> import matplotlib.pyplot as plt
>>> seg = shg.LineString([(0,0),(1,1)])
>>> fig = plt.figure()
>>> ax  = fig.gca()
>>> plot_line(ax,seg)
>>> plt.show()

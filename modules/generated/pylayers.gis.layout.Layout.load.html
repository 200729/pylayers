pylayers.gis.layout.Layout.load¶

Layout.load(_filename)[source]¶

load a Layout in different formats

Parameters:_filename : string

Notes

Available format are :

.ini : ini file format (natural one) DIRINI .str2 : native Pyray (C implementation) DIRSTRUC .str : binary file with visibility DIRSTRUC .osm : opens street map format DIROSM

layout files are stored in the directory pstruc[‘DIRxxx’]

pylayers.antprop.signature.Signatures.raysv¶

Signatures.raysv(ptx=0, prx=1)[source]¶

from signatures dict to 2D rays Vectorized version

Parameters:

ptx : numpy.array or int

Tx coordinates is the center of gravity of the cycle number if type(tx)=int

prx : numpy.array or int

Rx coordinates is the center of gravity of the cycle number if type(rx)=int
Returns:

rays : Rays

See also

Signatures.image, Signatures.backtrace

Notes

This is a vectorized version of Signatures.rays. This implementation take advantage of the np.ndarray and calculate images and backtrace for block of signatures. A block of signature gather all signatures with the same number of interaction.

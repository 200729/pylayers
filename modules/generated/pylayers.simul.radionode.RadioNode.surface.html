pylayers.simul.radionode.RadioNode.surface¶

RadioNode.surface(N1=2, N2=2, p0=[0, 0, 0], p1=[1, 0, 0], p2=[0, 1, 0], mode='subst')¶

add a surface to RadioNode

add a surface with basis (p0p1,p0p2)

Parameters:

N1 : int

default 2

N2 : int

default 2

p0 : array or list

first point

p1 : array or list

second point

p2 : array or list

third point

mode : string

>>> from pylayers.simul.radionode import *
>>> tx= RadioNode()
>>> tx.surface(10,10,[0,0,1.5],[3.0,0,1.5],[0.0,3.0,1.5],'subst')

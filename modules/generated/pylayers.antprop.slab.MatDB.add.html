pylayers.antprop.slab.MatDB.add¶

MatDB.add(name='MAT', cval=(1+0j), sigma=0, alpha_cmm1=1, mur=1, fGHz=1, typ='epsr')[source]¶

Add a material in the DB

Parameters:

name : string

material name :

cval : float or complex

epsilon or index :

sigma : float or complex

permeability :

mur : float

typ : string

{‘epsr’|’ind’|,’reim’,|’THz’} :

Notes

Different ways to enter a material are :

i) epsr : epsr and sigma cval = epsr sigma = sigma ii) ind : indice @ fGHz cval = indice iii) reim : real(epsr) and imag(epsr) @fGHz iv) THZ

Examples

>>> from pylayers.antprop.slab import *
>>> m = MatDB()
>>> m.load('matDB.ini')
>>> m.add('ConcreteJcB',cval=3.5+0*1j,alpha_cmm1=1.9,fGHz=120,typ='THz')
>>> m.add('GlassJcB',cval=3.5+0*1j,alpha_cmm1=1.9,fGHz=120,typ='THz')
>>> out = m.save('Jacob.ini')

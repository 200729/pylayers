pylayers.antprop.slab.MatDB.add¶

MatDB.add(**kwargs)[source]¶

add a material in the DB

Parameters:

name : string

material name

cval : float or complex

epsilon or index

sigma : float or complex

conductivity

mur : float

relative permeability

typ : string

{‘epsr’|’ind’|,’reim’,|‘THz’}

Notes

Different ways to enter a material are :

  1. ‘epsr’ : epsr and sigma

cval = epsr sigma = sigma

  1. ‘ind’ : indice @ fGHz

cval = indice

  1. ‘reim’ : real(epsr) and imag(epsr) @fGHz
  2. ‘THz’

Examples

>>> from pylayers.antprop.slab import *
>>> m = MatDB()
>>> m.load('matDB.ini')
>>> m.add(name='ConcreteJcB',cval=3.5+0*1j,alpha_cmm1=1.9,fGHz=120,typ='THz')
>>> m.add(name='GlassJcB',cval=3.5+0*1j,alpha_cmm1=1.9,fGHz=120,typ='THz')
>>> out = m.save('Jacob.ini')

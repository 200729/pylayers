pylayers.signal.bsignal.TUsignal.align¶

TUsignal.align(u2)[source]¶

align two Usignal on a same base

returns a list which contains the two aligned signals

It is assume that both signal u1 and u2 share the same dx This function can be improved regarding time granularity

Parameters:u2 : TUsignal
Returns:TUsignal y extended TU signal, x bases are adjusted

Examples

>>> import matplotlib.pylab as plt
>>> from pylayers.signal.bsignal import *
>>> i1 = TUsignal()
>>> i2 = TUsignal()
>>> i1.EnImpulse()
>>> i2.EnImpulse()
>>> i2.translate(-10)
>>> i3 = i1.align(i2)
>>> fig,ax=i3.plot(typ=['v'])
>>> plt.show()

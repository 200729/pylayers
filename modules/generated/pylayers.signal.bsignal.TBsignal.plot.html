pylayers.signal.bsignal.TBsignal.plot¶

TBsignal.plot(**kwargs)¶

Parameters:

Examples

>>> import numpy as np
>>> from pylayers.signal.bsignal import *
>>> from matplotlib.pylab import *
>>> x = np.array( [ 1, 3 , 6 , 11 , 18])
>>> y = np.array( [ 0,1 ,-5, 8 , 10])
>>> s = Bsignal(x,y)
>>> fi = figure()
>>> fig,ax=s.plot(typ=['v'])
>>> ti = title('TBsignal : plot')
>>> show()

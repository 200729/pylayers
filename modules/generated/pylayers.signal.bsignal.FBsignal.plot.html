pylayers.signal.bsignal.FBsignal.plot¶

FBsignal.plot(**kwargs)[source]¶

plot FBsignal

Parameters:

phase : boolean

default True

dB : boolean

default True

iy : index of y value to be displayed

default [0] only first the line is displayed

See also

Bsignal.plot

Examples

>>> from pylayers.signal.bsignal import *
>>> from numpy import *
>>> from scipy import *
>>> S = FBsignal()
>>> S.x = arange(100)
>>> S.y = cos(2*pi*S.x)+1j*sin(3*pi*S.x+pi/3)
>>> fig,ax = S.plot()
>>> plt.show()

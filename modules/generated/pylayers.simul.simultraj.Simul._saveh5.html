Notes

Dataset organisation:

simultraj_<trajectory_filename.h5>.h5

|time | ... | |/<tidx_ida_idb_wstd>/ |attrs | |a_k | |t_k

Root dataset : time : array

range of simulation time
Group identifier :
tidx : index in time dataset ida : node a index in Network idb : node b index in Network wstd : wireless standar of link interest
Inside group:
a_k : alpha_k values t_k : tau_k values

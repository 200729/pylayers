pylayers.util.geomutil.GeomVect.points¶

GeomVect.points(pt, colorname='blue')[source]¶

Geomview display a set of points with color

Parameters:

pt :

sequence of points np.ndarray or dictionnary whose value is a tuple (x,y,z)

colorname :

a colorname from coldict keys

Examples

>>> import numpy as np
>>> from pylayers.util.geomutil import *
>>> import scipy as sp
>>> pt1 = sp.rand(3,10)
>>> pt2 = { 1:(0,0,0),2:(10,10,10),3:(0,10,0),4:(10,0,0)}
>>> gv1 = GeomVect('test1')
>>> gv1.points(pt1)
>>> #gv1.show3()
>>> gv2 = GeomVect('test2')
>>> gv2.points(pt2)
>>> #gv2.show3()

pylayers.antprop.antenna.Antenna.Fpatt¶

Antenna.Fpatt(th=, []ph=, []pattern=True)[source]¶

generate antenna pattern

Parameters:

th : :

ph : :

pattern : boolean

default True

if pattern :

The pattern is generated from self.Nt and self.Np points

else: :

phi and theta have the same length (typically ray direction)

Examples

>>> from pylayers.antprop.antenna  import *
>>> A = Antenna(typ='Gauss')
>>> A.Fpatt()
>>> f,a = A.polar()
>>> plt.show()

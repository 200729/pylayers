Body.load(_filebody='John.ini')[source]¶

load a body ini file

Parameters:_filebody : body short filename

Notes

A body .ini file contains 4 sections

  • section [nodes]

Node number = Node name + section [cylinder] CylinderId = {‘t’:tail node number, ‘h’:head node number , ‘r’: cylinder’ radius} + section [antennas] Antenna number = {‘cylId’ cylinder Id, ‘l’: ,’h’: parameter,’a’:angle,’filename’: filename}

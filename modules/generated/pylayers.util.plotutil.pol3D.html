pylayers.util.plotutil.pol3D¶

pylayers.util.plotutil.pol3D(fig, rho, theta, phi, sf=False, shade=True, title='pol3D')[source]¶

polar 3D surface plot

Parameters:

rho : np.array

t x p

theta : np.array

1 x t

phi : np.array

1 x p

Examples

>>> from pylayers.util.plotutil import *
>>> import numpy as np
>>> theta = np.linspace(0,np.pi,90)
>>> phi = np.linspace(0,2*np.pi,180)
>>> rho = np.ones((len(theta),len(phi)))
>>> fig=plt.figure()
>>> pol3D(fig,rho,theta,phi)
>>> plt.show()

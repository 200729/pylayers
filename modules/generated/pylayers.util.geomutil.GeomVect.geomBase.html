pylayers.util.geomutil.GeomVect.geomBase¶

GeomVect.geomBase(M, pt=array([ 0., 0., 0.]), col=array([[0, 0, 1], [0, 1, 0], [1, 0, 0]]), linewidth=3, scale=1)[source]¶

Construct a geomview vect file for vizualisation of a frame

Parameters:

M : ndarray (3 x 3 )

[ v1, v2, v3 ]

pt : np.array

origin point (default (0,0,0))

col : :

color (3x3)

linewidth : :

linewidth (default 3)

Notes

by default the geomview filename is base0.vect

Examples

>>> from pylayers.util.geomutil import *
>>> import numpy as np
>>> v1 = np.array([1,0,0])
>>> v2 = np.array([0,1,0])
>>> v3 = np.array([0,0,1])
>>> M  = np.vstack((v1,v2,v3))
>>> #gv = GeomVect('test')
>>> #gv.geomBase(M)
>>> #gv.show3()

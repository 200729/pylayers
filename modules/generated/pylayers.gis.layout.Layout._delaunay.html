pylayers.gis.layout.Layout._delaunay¶

Layout._delaunay(poly, polyholes=[])[source]¶

make a Delaunay partitioning of a polygon

If polyhole == []

if a cycle is non convex

1- find its polygon 2- partition polygon into convex polygons (Delaunay) 3- try to merge partitioned polygons in order to obtain

the minimal number of convex polygons

If polyholes != []

polygon poly contains holes (polyholes)

This methods returns a partitioning of the polygon poly into several convex polygons (voronoi).

Parameters:

poly : sh.Polygon

polyhole : list of sh.Polygon
Returns:

ncpol : list

list of new created geu.Polygons

See also

pylayers.gis.layout.buildGt, pylayers.gis.layout.add_segment, pylayers.gis.layout.del_segment, pylayers.util.geomutil.Polygon, sp.spatial.Delaunay

Notes

The algorithm updates the Gt nodes and edges created into self.buildGt by adding new nodes and new _AIR segments.

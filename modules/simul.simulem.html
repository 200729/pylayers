cir(itx, irx, store_level=0, alpha=1.0, ext='', rep='output', format='a')[source]¶

Calculate a set of channel impulse responses

Parameters :

itx : transmitter index iterable set

irx : receiver index iterable set wav : applied waveform (st,sf,sfg)

this waveform includes the gamma factor

store_level : binary mask

bit 0 : save CVC bit 1 : save CVCO bit 2 : save CSCA bit 3 : save CIRo bit 4 : save CIRa

alpha : normalization factor ext : rep : format : string

a : with antenna o : omnidirectionnal

A factor ::math`sqrt{1}{30}` is necessary when applying the antenna

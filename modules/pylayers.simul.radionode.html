pylayers.simul.radionode¶

RadioNode Class¶

A radio Node is a data structure which store positions of a radio node at several time step.

This data structure is used in DLink.

class pylayers.simul.radionode.RadioNode(name='', typ='undefined', _fileini='radionode.ini', _fileant='defant.vsh3')¶

pylayers.network.communication¶

class pylayers.network.communication.Gcom(net=Network information ******************* number of nodes: 0 ID | name |group | type | position (x, y, z) | antenna | wstd -----------------------------------------------------------------------------------------, sim=<SimPy.Simulation.Simulation object>)[source]¶

Communication graph

Notes

That class inherits of networkx.MultiDiGraph(). Its nodes are the nodes of the passing Network object. Its edge represents all the possbile communication bridge between thoses nodes. On the edge, timestamped message can be passed between nodes

Contrary to pylayers.network, which is an undirected graph, here edge are directed.

Hence the dictionnary of simpy events (devt) can be created. The keys of devt are a tuple (nodeid#1, nodeid#2 , wstd).

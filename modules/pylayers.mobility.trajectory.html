Define a trajectory

This class derives from pandas.DataFrame. It handles a full 3D trajectory description.

A trajectory is time-stamped and contains information about position, velocity and acceleration.

Examples

>>> from pylayers.mobility.trajectory import *
>>> import matplotlib.pyplot as plt
>>> import numpy as np
>>> t = np.arange(0,10,0.01)
>>> x = 2*t*np.cos(t)
>>> y = 3*t*np.sin(t)
>>> z = 0*t
>>> pt =np.vstack((x,y,z)).T
>>> traj = Trajectory()
>>> traj.generate(t,pt)
>>> f,a = traj.plot()
>>> plt.show()

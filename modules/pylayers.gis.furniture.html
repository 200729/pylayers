Class Furniture

Notes

This class handle the description of furnitures. Until now furnitures are limited to rectangular objects. A possible evolution would be to authorize any polygon as a furniture element

Furniture are stored in an ini file of dirstruc.

from pylayers.gis.layout import * import matplotlib.pylab as plt L = Layout() L.loadstr(‘where1.ini’) L.showGs() L.loadfur(‘Furw1.ini’) fig = plt.gcf() ax = fig.get_axes()[0] for fur in L.lfur:

fur.show(fig,ax)

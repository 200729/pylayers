pylayers.antprop.slab¶

This module exploits heavily numpy broadcasting mechanism :

axis convention

  • nf : axis = 0 frequency axis
  • nt : axis = 1 angular axis
  • p : axis = 2 parallel axis
  • o : axis = 3 orhogonal axis

Layout ploting¶

The layout ploting exploits vizualization functions of networkx. The method showG allows to display the different graphs which constitute the layout. The different option for ploting a graph are {‘s’,’t’,’i’,’r’,’v’}

Slab name¶

Each node of with a negative index is a point.

Each node of with a positive index corresponds to a segment (Wall,door,window,...).

Each edge has a name attribute, which contains the name entry of its constitutive Slab.

In[74]:

Out[74]:

'PARTITION'

In the example above the node +20 of is a segment of nature ‘PARTITION’.

‘PARTITION’ is a key entry of the Slab dictionnary.

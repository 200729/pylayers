
pylayers
========

pylayers is a python platform for studying and testing localization algorithms 
in indoor environments. 

+ Indoor Layout description 
+ Mobile user mobility 
+ Indoor Propagation Simulation a Ray Tracing engine
+ Realistic description of antennas  
+ Handling of Ultra Wideband Waveforms 
+ Various techniques exploiting heterogeneous Radion observables

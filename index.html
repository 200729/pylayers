Python Indoor Propagation Simulator for Mobile Localization

PyLayers is an indoor mobile radio propagation simulator. Its purpose is to help designing heterogeneous localization (including Ultra Wideband Radios) for advanced communications systems and sensor networks

Features

  • Indoor layout edition
  • Separate modules for handling material,slab,antennas,channel,mobility,algorithms, ...
  • Handling of UWB waveforms
  • Motley Keenan multiwall model
  • Ray tracing tool based on layout graph extraction and evaluation of ray signatures
  • Agent mobility based on SimPy
  • Localization algorithms toolbox
  • Open source GPL license

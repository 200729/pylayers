Coverage with a Multi Wall model¶

Sometimes it might be useful to get a rough information about losses in a given indoor environment. Ray tracing is generallay quite time consuming and depending on the purpose of the study it could be sometimes also relevant to proceed with a simpler approach while stil site-specific. This is why PyLayers provides such a tool which heavily relies on the core module slab.py

First let’s import the coverage module

from pylayers.antprop.coverage import *
import time

Instantiate a coverage object. By defaut, TA-Office.str layout strucure is loaded.

This information can be modified in the coverage.ini file in the project directory.

Below is an example of how the content of the file looks like

Calculating Received Power Coverage¶

Then, the coverage calculation can be launched in calling the cover method

C.L.display['nodes']=False
C.L.display['ednodes']=False
C.L.showGs()

(<matplotlib.figure.Figure at 0x441d3d0>,
 <matplotlib.axes.AxesSubplot at 0x5245750>)

The shadowing map coverage results can be displayed by invoquing various functions.

  • showLoss : display the path loss
  • showPower : display the received power
C.fGHz=0.915
C.cover()
C.showPower()

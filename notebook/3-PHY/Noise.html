The bsignal module has a dedicated class for handling noise signal. To create a white noise just type :

>>> w = Noise()

The representation of the noise object provides information about default values. In digital representation of noise the sampling frequency is important. The noise signal is generated from a time to a time . The default power spectral density is and can be modified with the argument PSDdBmpHz.

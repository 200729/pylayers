Vector Spherical Harmonics Representation of Antennas¶

from pylayers.antprop.antenna import *

from pylayers.antprop.antvsh import *

%matplotlib inline

Loading an Antenna from a Matlab file

A = Antenna('S2R2.mat',directory='ant/UWBAN/Matfile')

The shape of the functions indicates :

  • :math:`N_{phi} = 180 `
np.shape(A.Fp)

(91, 180, 104)

The frequency array is expressed in and delays are expressed in 

fGHz = A.fGHz

fGHz.shape

(104,)

I = A.Ft[:,:,:]

I.shape

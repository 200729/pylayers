Control of the 4 axes ScannerÂ¶

The 4 axes scanner is controlled via the module smparker. To load this module :

>>> from pylayers.measures.parker.smparker import *

The scanner is connected to the control PC via a serial port. This control PC is itself connected to a Vector Network Analyzer via an ethernet connection.

For testing purpose, the scanner can be emulated.

S = Scanner(emulated=True)

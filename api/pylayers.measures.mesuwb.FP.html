FP¶

class pylayers.measures.mesuwb.FP(M, k, alpha=0.1, Tint=0.225, sym=0.25, nint=8, thlos=0.05, thnlos=0.15, thcum=0.15, w=6)¶

Bases: object

Fingerprint class

Rxid : 1 | 2 | 3 | 4 alpha : pdf percentile to suppress below and above (used in tau_rms and tau_moy) Tint : Integration time for Emax (ns) Tsym : Integration symetry default (0.25 | 0,75) nint : number of interval for toa_max thlos : threshold for los case toa_th thnlos : threshold for nlos case toa_th thcum : threshold for toa_cum

This method build the Fingerprint from UWBMeasure M - Rx number k

stepped_mode(sens=1, mode='STEP')¶

activate sweep mode

Parameters:

sens : int

mode : ‘stepped’

STEPped : source frequency is CONSTANT during measurement of eah displayed point. More accurate than ANALog. Dwell time can be set in this mode.
swept_mode(sens=1)¶

activate sweep mode

Parameters:

sens : int

ANALog : source frequency is continuously RAMPING during measurement of each

displayed point. Faster than STEPped. Sweep time (not dwell time) can be set in this mode.

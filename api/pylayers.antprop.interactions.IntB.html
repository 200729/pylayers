class pylayers.antprop.interactions.IntB(data=array([], dtype=float64), idx=[], slab={})¶

Notes

Examples

>>> from pylayers.antprop.rays import *
>>> M = np.eye(2).reshape(4)
>>> B = IntB(M,0)
>>> B.data
array([ 1.,  0.,  0.,  1.])
>>> B.stack(M,1)
>>> B.data
array([[ 1.,  0.,  0.,  1.],
       [ 1.,  0.,  0.,  1.]])
>>> eB=B.eval()
>>> nf = B.nf
>>> ninter = len(B.idx)
>>> np.shape(eB)
(1, 2, 2, 2)

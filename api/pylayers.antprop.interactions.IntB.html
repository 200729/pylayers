IntB¶

class pylayers.antprop.interactions.IntB(data=array([], dtype=float64), idx=[], slab={})¶

Bases: pylayers.antprop.interactions.Inter

Local Basis interaction class

Basis interactions

Notes

The interaction object is np.array with shape (nf,ninter 2, 2)

Attributes

data (np.array:) WARNING np.shape(data) = (ninter x 4) the input matrix 2x2 is reshaped as 1x 4
idx (list) index of the corresponding ray and interaction

Methods

Examples

>>> from pylayers.antprop.rays import *
>>> M = np.eye(2).reshape(4)
>>> B = IntB(M,0)
>>> B.data
array([ 1.,  0.,  0.,  1.])
>>> B.stack(M,1)
>>> B.data
array([[ 1.,  0.,  0.,  1.],
       [ 1.,  0.,  0.,  1.]])
>>> eB=B.eval()
>>> nf = B.nf
>>> ninter = len(B.idx)
>>> np.shape(eB)
(1, 2, 2, 2)

Interactions¶

class pylayers.antprop.interactions.Interactions(slab={})¶

Bases: pylayers.antprop.interactions.Inter, dict

Interaction parameters

gather all type of interactions (IntB/L/R/T)

Methods

add(self,li): add a list of basis interactions  
addi(self,i): add a single interaction  
eval(self) (evaluate all the interactions added thanks to self.add or self.addi) and create the self.I which gather all thoses interactions
5 following types of interactions  
B (local basis transformation matrix (unitary))
L (LOS case)
R (Reflection)
T (Transmission)
D (Diffraction)
